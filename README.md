# 蔚蓝档案插件
***
## 基于yunzai-bot的一款蔚蓝档案插件
目前使用的是[arona](https://doc.arona.diyigemt.com)的数据，其他功能一个没加（我是菜鸟）.
***
命令：ba攻略XXX 例：

ba攻略大叔

ba攻略国际服总战力

ba攻略国际服未来视

ba攻略11-1

ba攻略11h-1

ba插件更新 #ba强制更新

ba帮助

***
使用方法

1、在yunzai根目录下进入git或者cmd窗口复制以下代码

```javascript
git clone https://github.com/BlackLK/stuka-blue_archive.git ./plugins/stuka-blue_archive
```

2、重启yunzai机器人
